# OI_Progress
Helper that keep track of OIs

## crawled
- [evaluator.thailandoi](https://evaluator.thailandoi.org)
- [oichecklist](http://oichecklist.pythonanywhere.com)
- [codeforces](https://codeforces.com/)

## to use 
1. Download the file
2. install dependencies `pip3 install -r req`
3. move **oiprog** to `PATH`
4. enjoy running **oiprog**, starting with `oiprog help`
